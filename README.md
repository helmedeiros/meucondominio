# meucondominio

A small PHP pet project exploring a **condominium management** site.

An early hands-on experiment with PHP: a couple of pages (a landing page and a
registration form backed by `cadastro.php`) for managing a residential
condominium. Kept as a learning artifact.

## Pages

| File           | Purpose                          |
| -------------- | -------------------------------- |
| `index.html`   | Landing page.                    |
| `cadastro.php` | Registration form handler.       |
| `picture.php`  | Image handling.                  |

Static assets live under `images/`.
